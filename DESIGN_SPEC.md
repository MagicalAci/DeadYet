# 🎨 「还没死？」完整设计规范

## 📱 一、App Icon（必须）

| 字段 | 内容 |
|------|------|
| **资源名称** | App Icon 应用图标 |
| **尺寸** | 1024×1024 px |
| **格式** | PNG（无透明背景） |
| **文件名** | `icon-1024.png` |
| **是否必须** | ✅ 必须 |

### 设计规范
- **圆角**：不需要手动加圆角，iOS会自动处理
- **背景**：必须是不透明的纯色或渐变，不能有透明区域
- **安全区域**：主要内容应在中心 80% 区域内，避免被圆角裁切
- **清晰度**：在缩小到 60×60 时仍能辨认

### 视觉描述
> 一个简洁有力的图标，传达"打工人下班签到"的核心概念。
> 
> **推荐方案A**：深黑背景 (#1C1C1E)，中间是一个绿色对勾 ✓ (#34C759)，对勾下方有一条红色的心电图线（象征"还活着"）
> 
> **推荐方案B**：深黑背景，一个可爱但疲惫的牛头/马头剪影，眼睛是两个下班打卡的绿点
> 
> **推荐方案C**：渐变背景（深红到黑），中间是艺术化的"活"字或骷髅头戴着工牌

### 情绪关键词
`黑色幽默` `疲惫但倔强` `打工人自嘲` `下班的希望` `存活确认`

## 🎨 全局设计规范

### 配色系统

| 颜色名称 | 色值 | 用途 |
|----------|------|------|
| 警告红 DeadRed | `#FF3B30` | 加班状态、危险提示、强调 |
| 存活绿 AliveGreen | `#34C759` | 已下班、成功、存活确认 |
| 深黑背景 DarkBG | `#1C1C1E` | 主背景色 |
| 金色 Gold | `#FFD700` | 锦旗、成就、高级 |
| 银色 Silver | `#C0C0C0` | 次级成就 |
| 铜色 Bronze | `#CD7F32` | 初级成就 |
| 钻石蓝 Diamond | `#00CED1` | 顶级成就 |
| 灰色文字 GrayText | `#8E8E93` | 次要文字 |

### 字体建议
- **中文**：苹方 (PingFang SC) 或思源黑体
- **英文/数字**：SF Pro Display / SF Pro Text
- **标题**：粗体 Bold
- **正文**：常规 Regular

### 风格关键词
`暗黑系` `黑色幽默` `打工人文化` `自嘲` `简洁有力` `情绪共鸣`
