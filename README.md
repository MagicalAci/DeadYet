# 🐂🐴 还没死？(DeadYet)

> **面向打工人的毒舌下班签到APP**
> 
> 每天下班，报个平安，骂骂这个狗屎世界

---

## 🎯 产品定位

### 核心用户画像
- **25-35岁职场人士**（互联网、金融、广告等996重灾区）
- **心态**：自嘲、黑色幽默、渴望发泄
- **痛点**：加班文化压抑、缺乏情绪出口、需要群体共鸣

### 情绪设计

| 情绪层级 | 设计策略 | 实现方式 |
|---------|---------|---------|
| **发泄** | 让用户骂出来 | 语音/文字抱怨输入 |
| **共情** | 你不是一个人 | 地图显示全国牛马分布 |
| **刺激** | AI骂醒你 | 毒舌AI回复 + 招聘推荐 |
| **成就** | 游戏化存活 | 锦旗系统 + 存活天数 |
| **紧迫** | 战场氛围 | 实时推送战况 |

#### 签到流程
1. **首次使用** → 输入工作邮箱（无需注册）
2. **点击签到** → 可选输入抱怨（文字/语音）
3. **AI处理** → 生成锦旗 + 毒舌回复
4. **完成** → 存活天数+1

#### 锦旗系统设计

| 存活天数 | 锦旗称号 | 锦旗样式 |
|---------|---------|---------|
| 1-7天 | 新鲜韭菜 | 🌱 绿色基础锦旗 |
| 8-30天 | 牛马新星 | ⭐ 铜色锦旗 |
| 31-90天 | 资深社畜 | 🏅 银色锦旗 |
| 91-180天 | 钢铁打工人 | 🎖️ 金色锦旗 |
| 181-365天 | 不死老兵 | 👑 钻石锦旗 |
| 365+天 | 传奇牛马 | 💎 彩虹锦旗 |

## 🤖 AI功能设计

### AI人格设定

**角色名**：毒舌张
**性格**：
- 说话直接、不留情面
- 表面在骂你，实际在关心你
- 会推荐出路（招聘网站、摸鱼技巧）
- 偶尔冒出金句让你破防

### AI回复模板

```javascript
// 抱怨类型识别 → 对应回复模板

// 1. 加班相关
"又加班？你这是打工还是卖身？去Boss直聘逛逛，说不定有条活路。"
"加班费呢？没有？那你加个屁，明天摸鱼摸回来！"

// 2. 领导相关  
"你领导是不是脑子有坑？建议录音，以后仲裁用得上。"
"这种傻逼领导全国多了去了，你不走他走不了，懂？"

// 3. 同事相关
"职场没有朋友，只有利益。清醒点，干活去。"
"让他去死，你继续苟着，熬到比他先跑路。"

// 4. 工资相关
"就这点钱你还干？我真服了你这种老实人。"
"穷是暂时的，被压榨是持久的。跳啊，怂什么？"

// 5. 通用鼓励（毒舌版）
"行吧，又活过一天，明天继续被操。睡吧傻逼。"
"恭喜你没猝死，这就是你今天最大的成就。"
```

### 推送通知设计

| 时间段 | 频率 | 推送内容风格 |
|-------|------|-------------|
| 18:00-21:00 | 每小时 | 战况播报（轻松） |
| 21:00-23:00 | 每30分钟 | 紧急催促（焦虑） |
| 23:00-02:00 | 每15分钟 | 深夜关怀（毒舌+温情） |

#### 推送文案示例

```
// 18:00
🎉 战报：北京海淀区 已有 2,341 名牛马成功撤离！你还在吗？

// 19:00

// 21:00
⚠️ 警告：现在还没下班的都是重度社畜，建议自我反省

// 22:00
🚨 紧急：你的同行们都回家了，你还在公司？公司给你发老婆了吗？

// 23:00
💀 深夜档：还有 12,847 个可怜人没下班，你们是加班还是住公司？

// 00:00
🌙 午夜特报：还在加班？你不要命了？明天见（如果还活着的话）
```

## 🔧 技术架构

### 前端 (iOS)
- **语言**：Swift 6
- **框架**：SwiftUI
- **最低版本**：iOS 18
- **依赖**：
  - MapKit (地图)
  - Speech (语音识别)
  - UserNotifications (推送)

### 后端 (Zeabur)
- **运行时**：Node.js 20
- **框架**：Hono (轻量级)
- **数据库**：PostgreSQL
- **缓存**：Redis
- **AI**：OpenAI API / Claude API

### API设计

```
POST   /api/auth/email       # 邮箱验证
POST   /api/checkin          # 签到
GET    /api/checkin/history  # 签到历史
GET    /api/map/stats        # 地图统计
GET    /api/map/complaints   # 抱怨列表
POST   /api/complaints       # 发布抱怨
POST   /api/ai/roast         # AI毒舌
GET    /api/push/status      # 推送状态
```
